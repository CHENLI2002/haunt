# @haunt/ui

Design tokens shared by `apps/web` and `apps/mobile` — the single source of truth for color, type, spacing, radius (design-system skill).

> ⚠️ Brand visual spec is **TBD**. `tokens.ts` ships neutral placeholders; do not treat the colors as final. When the brand lands, update tokens here and the [design-system skill](../../.claude/skills/design-system/SKILL.md).

Components reference `semantic.*` tokens, never raw hex/px.
